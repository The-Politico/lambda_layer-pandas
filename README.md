# lambda_layer-pandas

An [AWS Lambda layer](https://docs.aws.amazon.com/lambda/latest/dg/configuration-layers.html).

##### Runtimes

- python3.6

##### Dependencies

- numpy
- pandas

### ARN

```
arn:aws:lambda:us-east-1:041537238831:layer:lambda_layer-pandas:1
```

### Building

```
$ make build
$ make deploy
```
